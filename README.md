# Generic-Singly-Linked-List
Use this library to create a generic singly linked list.

Helper Functions:
1. insert(data [, position])
2. remove(element)
3. print(void)
4. sort(void)
5. len(void)
6. insert_beg(data)
7. insert_last(data)

Library uses merge sort to sort the linked list.
